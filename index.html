Scripts AD & Config Linux

Bienvenue sur le repository dédié aux scripts pour Active Directory et aux fichiers de configuration Linux.
Vous trouverez ici des outils pour automatiser votre gestion AD ainsi que des fichiers pratiques comme .nanorc pour améliorer votre environnement Linux.

# Exemple d'un script bash pour configurer nanorc

echo "set tabsize 4" >> ~/.nanorc
echo "set linenumbers" >> ~/.nanorc
